# 莊子思想地圖

本圖以《莊子》現有篇章為中心，連結原典、注家解讀與本書現代詮釋。箭頭表示可互相照明，並非嚴格的推論次序。

```text
                                 [道]
              〈齊物論〉、〈大宗師〉、〈天道〉
                                   |
          +------------------------+------------------------+
          |                        |                        |
      [逍遙／自由]              [齊物／變化]              [養生／技道]
       〈逍遙遊〉                〈齊物論〉                〈養生主〉
          |                        |                        |
   有待 <-> 無待             是非的成心                 庖丁／依乎天理
          |                        |                        |
   無己、無功、無名          物化（夢蝶）              緣督以為經
          |                        |                        |
   無用之用 <------------------+------------------------> 處世的分寸
 〈逍遙遊〉、〈人間世〉        卮言／寓言                 〈人間世〉
          |                        |                        |
          +------------ [工夫與人格] ----------------------+
                         心齋、坐忘、真人
                      〈人間世〉、〈大宗師〉
                               |
             +-----------------+------------------+
             |                                    |
      [死亡與變化]                          [政治與關係]
  安時處順、氣化、物化                  無為、因任、應物
 〈大宗師〉、〈至樂〉                  〈人間世〉、〈應帝王〉
```

## 閱讀路徑

1. **自由的問題**：由〈逍遙遊〉讀有待、無待與逍遙；再讀無用之用，避免把自由誤作擁有更多資源。
2. **判斷與衝突**：由〈齊物論〉讀成心、是非與物化；〈寓言〉的卮言說明如何避免把語言變成封閉判決。
3. **工夫與處世**：由〈人間世〉的心齋進入高風險關係，再以〈大宗師〉坐忘、真人理解自我鬆動。
4. **生活的技藝**：由庖丁解牛讀〈養生主〉，把「順其固然」理解為辨識結構、節制用力，非神祕捷徑。
5. **生死與治理**：〈至樂〉、〈大宗師〉提供變化觀；〈應帝王〉、〈在宥〉則追問權力如何少以私意擾民。

## 重要交叉連結

- 「無待」與「無用之用」：前者檢查自由是否受條件挾制，後者反省功利尺度是否過窄。
- 「齊物」與「卮言」：前者鬆動絕對的是非，後者提供可隨情境調整的言說方式。
- 「心齋」與「坐忘」：皆非放棄心智；前者偏向進入關係前的虛心，後者更突出鬆開形知與身分。
- 「真人」與「死亡」：真人不是無情，而是在生死變化中不以恐懼、名利完全支配自己。
