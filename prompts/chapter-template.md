# 篇章寫作提示詞（出版版）

你正在撰寫《莊子全解》的一篇完整解析。請嚴格遵守 `AGENTS.md`。

## 輸入

- 篇名：〈{{TITLE}}〉
- 部類：{{PART}}
- 檔案：`content/chapters/.../{{ID}}-{{SLUG}}.md`
- 版本目標：先求 **draft 可讀**，再升 **review**

## 輸出要求

1. 保留 YAML frontmatter，更新 `status`、`updated`、`summary`（若更精準）。
2. 依 01–17 節完整撰寫；不得刪節標題。
3. **原典**只引用必要段落，並註明版本（預設郭慶藩《莊子集釋》系統）。
4. **白話**忠於文意，避免過度白話到失去古典張力。
5. **段落解析**必須回答：這段為什麼在這裡？與前後文如何銜接？
6. **注家**至少處理：郭象、成玄英；能加林希逸或其他則加。
7. **哲學分析／現代應用**標明為現代詮釋。
8. **與佛學比較**：與老子、儒家節同樣作多角度對照即可；點到概念異同，不必反覆聲明「不是佛學」。無可靠對應時可簡寫，勿硬比附。
9. 文末填寫交叉引用清單。
10. 產出結構圖與心智圖（ASCII 或 Mermaid）。

## 品質檢核（寫完自問）

- [ ] 讀者能否不靠聊天紀錄、只靠本檔讀懂本篇？
- [ ] 是否區分原典／注家／詮釋？
- [ ] 是否貼近原文順序而非主題拼貼？
- [ ] 現代應用是否回扣概念？
- [ ] 是否有延伸閱讀？

## 禁止

- 把「我認為」寫成「莊子認為」而無原文依據
- 只用心靈雞湯句子充篇幅
- 抄襲今人譯注原文而不標明
- 為了完整而灌水重複
